Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in DOM among them we have learned about get a element by id, Get a element by class, Select a element by tag also we have learned about querySeletor and querySelectorAll as well.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHtml is the full part of the HTML code which is content of inside of an element. On the other hand, innerText is the visible text inside an element and textContent indicates is all the text hidden or shown

Question-3: What is event delegation in the DOM?

Event delegation is use to add a event listener to a parent element instead of adding for all the child element. This event allow to handle event for all its children.

Question-4: What is event bubbling in the DOM?

Event bubbling is when we start a event from a element and moves up for its parent element and its grandparent element until we reach the root which is document.

Question-5: How do you create, add, and remove elements using JavaScript?

In javaScript we use document.createElement() for creating element, we use append() to add a element and remove() for remove a element.
